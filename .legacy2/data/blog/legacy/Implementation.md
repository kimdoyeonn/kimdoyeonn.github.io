# 구현

머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정

문제 해결 분야에서 구현 유형의 문제는 '풀이를 떠올리는 것은 쉽지만 소스코드로 옮기기 어려운 문제'를 의미한다.

대체로 사소한 조건 설정이 많은 문제일수록 코드로 구현하기 까다롭다.

- 완전탐색: 모든 경우의 수를 주저없이 다 계산하는 해결 방법
- 시뮬레이션: 문제에서 제시한 알고리즘을 한 단계식 차례대로 직접 수행
