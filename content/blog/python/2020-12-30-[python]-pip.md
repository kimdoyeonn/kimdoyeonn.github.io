pip는 파이썬으로 작성된 패키지 소프트웨어를 설치, 관리하는 패키지 관리 시스템이다. 파이썬 3.4 이후에 버전에는 같이 포함되어있다.

### 패키지 설치하기

#### 장고 설치

```powershell
pip install Django==3.1.4
```
