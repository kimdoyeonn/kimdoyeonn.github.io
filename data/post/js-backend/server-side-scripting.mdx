서버 측 스크립팅은 웹 서버에서 실행되는 스크립트 언어를 통해 동적인 웹 페이지를 생성하는 프로세스를 말한다.

서버 측 스크립트는 클라이언트 측 스크립트와 대조되며, 서버에서 실행되어 동적인 콘텐츠를 생성하거나 데이터베이스와 상호 작용하는 등의 작업을 수행한다.

대표적인 서버 측 스크립팅 언어는 PHP, Node.js(JavaScript), Ruby on Rails, Python(Django, Flask)이 있다.
